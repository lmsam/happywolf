# 終極一夜狼人殺 (One Night Ultimate Werewolf) - 遊戲說明書

## 1. 遊戲簡介
這是一個快節奏的推理遊戲，每局遊戲只有一個晚上和一個白天。玩家需要在有限的時間內，透過角色的能力獲取資訊，並在白天通過討論找出狼人。

## 2. 遊戲流程

### 2.1 設定階段 (Setup)
1.  **玩家入座**：輸入玩家名稱。
2.  **建立牌庫**：選擇參與遊戲的角色卡牌。
    *   卡牌數量 = 玩家人數 + 3 (其中 3 張為「中央牌」)。
3.  **確認身份**：
    *   每位玩家隨機抽取一張卡牌。
    *   **眼牌階段 (Peek Phase)**：玩家輪流查看自己的初始身份，然後蓋上。
    *   *注意：如果場上沒有狼人玩家，系統會強制交換一張狼人牌給隨機玩家 (Force Werewolf 規則)。*

### 2.2 黑夜階段 (Night Phase)
所有玩家閉眼，系統會依序叫醒有功能的角色進行行動。
*   **行動時間**：每個角色有固定 10 秒行動時間。
*   **假回合 (Fake Turn)**：即使該角色不在場上（在中央牌堆），系統仍會播放語音並倒數，以混淆視聽。

### 2.3 白天階段 (Day Phase)
天亮了！所有玩家睜開眼睛。
1.  **自由討論**：玩家根據晚上的資訊進行討論、推測或欺騙。
2.  **倒數計時**：預設 5 分鐘 (可調整)。

### 2.4 投票與揭曉 (Vote & Reveal)
1.  **投票**：倒數結束後，主持人倒數「3、2、1」，所有玩家同時指向自己懷疑是狼人的玩家。
2.  **處決**：得票最多的玩家被處決。
    *   如果有玩家得票數相同，則同時被處決。
    *   如果沒有人得票 (或平票且每人 1 票)，則無人被處決。
3.  **揭曉身份**：翻開所有卡牌，結算勝負。

## 3. 勝利條件
*   **村莊陣營勝利**：
    *   至少有一名 **狼人** 被處決。
    *   如果場上沒有狼人 (都在中央)，則 **無人被處決** 時村莊勝利。
*   **狼人陣營勝利**：
    *   沒有狼人被處決。
*   **特殊勝利**：
    *   **皮匠 (Tanner)**：如果皮匠被處決，皮匠獨自勝利。
    *   **小兵 (Minion)**：如果狼人存活，小兵與狼人共享勝利；如果小兵被處決但狼人存活，小兵也算贏 (視乎具體規則，本 APP 判定狼人存活即狼人陣營勝)。

## 4. 角色能力詳解 (按喚醒順序)

| 順序 | 角色 | 陣營 | 能力描述 |
| :--- | :--- | :--- | :--- |
| 1 | **化身幽靈 (Doppelgänger)** | 特殊 | 查看一位玩家的牌，並**複製**該角色的能力。如果複製到有夜間行動的角色，會立即執行該角色的行動。 |
| 2 | **狼人 (Werewolf)** | 狼人 | 確認彼此身份。如果是**孤狼** (只有一隻狼人)，可以查看一張中央牌。 |
| 3 | **小兵 (Minion)** | 狼人 | 看見誰是狼人，但狼人不知道你是誰。 |
| 4 | **守濟會 (Mason)** | 村莊 | 互相確認身份。 |
| 5 | **預言家 (Seer)** | 村莊 | 查看一位玩家的牌，**或者** 查看兩張中央牌。 |
| 6 | **強盜 (Robber)** | 村莊 | 查看並**交換**自己與另一位玩家的牌。 |
| 7 | **搗蛋鬼 (Troublemaker)** | 村莊 | 交換另外兩位玩家的牌 (不能看牌)。 |
| 8 | **酒鬼 (Drunk)** | 村莊 | 將自己的牌與一張中央牌交換 (不能看換回來的牌)。 |
| 9 | **失眠者 (Insomniac)** | 村莊 | 查看自己最後的牌 (確認是否被換過)。 |
| - | **村民 (Villager)** | 村莊 | 無特殊能力。 |
| - | **獵人 (Hunter)** | 村莊 | 如果被處決，被他指名的玩家也會被處決。 |
| - | **皮匠 (Tanner)** | 特殊 | 你的目標是被處決。 |

## 5. 特殊機制說明
*   **初始身份 vs 當前身份**：夜間行動是根據玩家**一開始抽到的身份** (Initial Role) 進行喚醒。即使你的牌在夜間被強盜或搗蛋鬼換走了，你依然會被原本的角色喚醒。
*   **化身幽靈標記**：如果某個角色的能力是由化身幽靈發動的，揭曉時該卡牌左上角會有 `🎭` 標記。
