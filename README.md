# One Night Ultimate Werewolf Moderator App

A web-based moderator application for the board game "One Night Ultimate Werewolf" (終極一夜狼人殺).

## Features

-   **Game Logic**: Manages the Night Phase sequence and Day Phase timer.
-   **Bilingual Support**: English and Traditional Chinese (Cantonese).
-   **Text-to-Speech**: Announces roles and instructions.
-   **Interactive Table**: Grid layout for managing cards and players.
-   **Role Library**: Select roles to include in the game.
-   **Visuals**: Dark theme with "Neon Arcade" / Shonen Manga aesthetic.

## Usage

1.  Open `index.html` in a modern web browser.
2.  Select the roles for the game.
3.  Add player names (optional).
4.  Click "Start Game".
5.  Follow the on-screen and audio instructions.

## Development

-   **Tech Stack**: HTML, CSS, Vanilla JavaScript.
-   **Assets**: Images stored in `images/`.
